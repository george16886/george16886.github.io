Linux Terminal Notes


Alias Setting

%> alias ll='ls -a'

You can check what alias is currently configured by

%> alias

To make the alias permanent, add command into ~/.bashrc by

%> echo "alias ll='ls -a'" >> ~/.bashrc

or directly edit the file, then

%> source ~/.bashrc

Connect to PTT

%> ssh bbsu@ptt.cc

  • No encryption using telnet.

  • In addition to security, SSH performs data compression on tranmission, accelerating data transmission rate.

  • The common encoding of bbs is using Big5, PTT is using Big5-UAO.

  • Mojibake will show if ssh bbs@ptt.cc, ssh bbsu is for UTF-8.

  • SSH = Secure SHell

  • BBS = Bulletin Board System

  • UAO = Unicode-At-On

  • UTF-8 = 8-bit Unicode Transformation Format

Telnet is a protocol used on the Internet or local area networks to provide a bidirectional interactive text-oriented communication facility using a virtual terminal connection. User data is interspersed in-band with Telnet control information in an 8-bit byte oriented data connection over the Transmission Control Protocol (TCP).

SSH is a cryptographic network protocol for operating network services securely over an unsecured network. The best known example application is for remote login to computer systems by users.


Advanced Packaging Tools (APT)

%> apt-get -h 
%> apt-cache -h

Show a short usage summary.

%> apt-cache search xxx

Keyword-search for packages.

%> sudo apt-get update

Re-synchronize the package index files from their sources. The indexes of available packages are fetched from the locations specified in /etc/apt/sources.list (package resource list for APT).

%> sudo apt-get (dist-)upgrade

Install the newest versions of all packages currently installed on the system from the sources enumerated in /etc/apt/sources.list.

Using dist-upgrade will attempt to upgrade the most important packages at the expense of less important ones if necessary, which is more risky.

If a new version of a package needs a new dependency, apt-get dist-upgrade will upgrade and install the dependency, apt-get upgrade won’t.

Be sure to perform an update before an upgrade or dist-upgrade.

%> sudo apt-get install/remove {pkg}

Install or remove packages.

%> sudo add-apt-repository ppa:<user>/<ppa_name>

Adds a repository into the /etc/apt/sources.list or /etc/apt/sources.list.d/xxx.list.

%> sudo --remove add-apt-repository ppa:<user>/<ppa_name>
%> sudo -r add-apt-repository ppa:<user>/<ppa_name>

Removes an existing repository. You can also remove PPAs by deleting the .list files from /etc/apt/sources.list.d directory.

As a safer alternative, you can install ppa-purge:

%> sudo apt-get install ppa-purge

And then remove the PPA, downgrading gracefully packages it provided to packages provided by official repositories:

%> sudo ppa-purge ppa_name

Note that this will uninstall packages provided by the PPA, but not those provided by the official repositories. If you want to remove them, you should tell it to apt:

%> sudo apt-get purge package_name
  • PPA = Personal Package Archives

Install Third-party Packages

  1. by PPA (Personal Package Archive)

    First add third-party PPA into package resource list:

    %> sudo add-apt-repository ppa:{pkg}

    Then update and intall:

    %> sudo apt-get update
%> sudo apt-get install {pkg}

  2. by .deb file (file format for the Debian distribution)

    %> sudo dpkg -i xxx.deb

    In case dpkg indicates missing dependencies, complete the installation forcibly by

    %> sudo apt-get install -f

GVIM (GUI-Vim)

Installation:

%> sudo apt-get install vim  
%> sudo apt-get install vim-gtk  

%> vim --version

Some useful settings:

%> vim ~/.gvimrc

color desert       //set color scheme

You can also change those settings in VIM by

:set xxx
:set               //show current settings
